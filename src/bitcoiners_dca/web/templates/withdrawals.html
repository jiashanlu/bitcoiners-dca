Safety

  • Use a fresh address per exchange from your hardware wallet, not the same one twice.
  • Trade-only API key is enough — the exchange's withdraw whitelist authorizes the address, not the bot.
  • Set each exchange's withdraw whitelist to ONLY the destination here before turning auto-withdraw on. Belt + braces.
  • Lightning withdrawals are only supported by OKX (BitOasis and Binance UAE are on-chain only).
